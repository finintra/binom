# 🧮 Біном Ньютона — для дітей

Інтерактивна презентація, яка пояснює формулу **(a + b)ⁿ** простою та веселою мовою для дітей 10–13 років.

🌐 **Дивитись онлайн:** https://finintra.github.io/Binom/

## Що всередині

**17 слайдів** з котиком-помічником Біником:

1. Привітання
2. Що таке (a + b)?
3. Пояснення на прикладах з яблуками
4. Чому просто множити не можна?
5. Квадрат біному — візуалізація через коробку з кубиків
6. Підрахунок шматочків: a² + 2ab + b²
7. Формула (a + b)²
8. Куб біному — 3D-візуалізація
9. Формула (a + b)³
10. Чарівний трикутник Паскаля
11. Навіщо це все в житті?
12. Вправа: (a + b)⁴
13. 🎮 **Інтерактивна гра** — 3 питання з вибором відповіді
14. Цікавий факт про Ньютона (йому було 22!)
15. До побачення

## Як керувати

- **← →** або **Space** — перехід між слайдами
- **Клік** — розгорнути анімацію на слайді
- **1–9** — стрибнути на слайд
- **R** — на початок
- **?** — довідка

## Технології

- Чистий HTML + CSS + JavaScript (без фреймворків)
- Web Component `<deck-stage>` для керування слайдами
- Кастомні SVG-ілюстрації (котик Біник)
- Автомасштабування 1920×1080 під будь-який екран

## Файли

- `index.html` — головна сторінка (для GitHub Pages)
- `Біном Ньютона.html` — те саме, з локалізованою назвою
- `styles.css` — стилі
- `cat.js` — SVG-ілюстрації котика
- `deck-stage.js` — движок презентації

## Розгортання на GitHub Pages

1. Залити всі файли в репозиторій
2. Налаштування → Pages → Source: `main` branch, папка `/ (root)`
3. Зачекати ~1 хвилину
4. Відкрити `https://<username>.github.io/<repo>/`
